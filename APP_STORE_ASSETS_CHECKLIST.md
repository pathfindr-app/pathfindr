# App Store Assets Checklist

This is a practical checklist you can use while preparing iOS and Android store listings.

## Shared
1. App name and tagline
2. App icon (1024x1024)
3. Privacy Policy URL
4. Terms of Service URL
5. Support email

## iOS (App Store Connect)
1. Screenshots
- iPhone 6.7"
- iPhone 6.1"
- iPhone 5.5" (if required)

2. App preview video (optional but recommended)
3. Subtitle
4. Keywords list
5. Category and age rating
6. Privacy nutrition labels
7. Sign in with Apple if any third-party login exists

## Android (Google Play Console)
1. Feature graphic (1024x500)
2. Phone screenshots
3. 7" and 10" tablet screenshots (recommended)
4. Short description (80 chars)
5. Full description (4000 chars)
6. Data Safety form

## In-App Purchase Setup
1. Store products created
- iOS IAP product ID matches RevenueCat
- Android product ID matches RevenueCat
2. Entitlements mapped in RevenueCat
3. Test purchases on device

## Ads Setup
1. AdMob production IDs
2. Test mode disabled for release builds
3. Confirm Pro removes ads
